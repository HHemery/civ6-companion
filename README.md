# Civ Sheet

Mémo compétitif Civ VI (mod **BBG**) : guides early/mid/late game, tier list, bilan de
tips, **encyclopédie BBG** (leaders, unités, merveilles, doctrines, cités-états…) et une
fiche de civ par joueur.

- **Site en ligne** : https://hhemery.github.io/civ6-companion/
- **En local** : double-clic sur `index.html` (aucune dépendance, aucun build).

## 🏆 Ajouter ta civ (pour les amis)

Tu veux ton espace sur le site, avec ta civ et ton portrait ? Suis
**[AJOUTER-SA-CIV.md](AJOUTER-SA-CIV.md)** : ton propre Claude génère la fiche depuis une
transcription, tu ouvres une Pull Request, Hugo la merge. (Un onglet **« Ajouter ta civ »**
explique aussi tout ça directement sur le site.)

## 🛠️ Comprendre / faire évoluer le repo

Lis **[CLAUDE.md](CLAUDE.md)** (règles d'or) et **[PROJET.md](PROJET.md)** (architecture
complète, schéma des données, conventions).
